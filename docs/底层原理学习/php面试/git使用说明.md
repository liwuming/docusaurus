# git是什么
git是一个分布式的代码版本控制工具


git init
git clone
git pull 
git push 





文件的三种状态
已修改--modified
已暂存--staged
已提交--committed



工作区--是对项目的某个版本独立提取出来的内容
暂存区--保存了下次将要提交的文件列表信息
git仓库目录--是git用来保存项目的元数据和对象数据库的地方，


安装git
git --version


安装git之后的第一件事，是对git的配置
git config --global user.name "liwuming"
git config --global user.email 1459543371@qq.com


检查配置信息
git config --list




获取git仓库
通常有两种获取git项目仓库的方式
- 将一个已存在的目录
- 克隆已存在的git仓库

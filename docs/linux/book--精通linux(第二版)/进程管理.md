进程与线程

线程是指进程中的一个执行流程，执行场景

当进程内的多个线程同时运行时，这种运行方式成为并发运行，如数据库服务器和web服务器都支持并发运行--同时相应来自不通客户端的请求


进程


进程和线程的主要区别在于：每个进程都需要操作系统为其分配独立的内存地址空间，而同一进程中的所有线程在同一块地址空间中工作，这些线程可以共享同一块内存和系统资源，比如共享一个对象或者共享已经打开的一个文件。

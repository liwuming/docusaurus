

# 索引是什么









使用索引的代价

索引也需要占用一定的磁盘空间，

myisam存储引擎而言，数据与索引是独立存储的，
innodb存储引擎，数据与索引是一起存储的
使用索引可以加速查询速度，但是会影响插入和更新的速度






innodb存储引擎支持两种常见的索引
- B+tree索引
- 自适应哈希索引

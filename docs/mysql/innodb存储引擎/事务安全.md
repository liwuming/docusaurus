# 事务的控制语句
> 在mysql命令行的默认设置下，事务都是自动提交的，

- begin 
- start transction
- set autocommit=0;


关于事务的配置
autocommit：设置事务的提交方式
0：手动提交
1：自动提交（默认值）

transaction：设置事务的隔离级别
- read uncommit
- read commit
- repay read
